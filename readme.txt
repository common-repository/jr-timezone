== Description ==

This timezone plugin allows your users to find the current timezone in a variety of cities and countries - as a widget on your blog!

All you need to do is specify a title in the JR Timezone settings and then enable the widget! The plugin will now be active!


== Installation ==

1. Upload `/jr-timezone.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the JR Timezone Settings in your Administration Panel
4. Done!
